# RPS+
## A Realtime WebSockets Based Rock , Paper and Scissors Game

* TeckStack
* Frontend - Next.js , TerboRepo , Shadcn , Typescript
* Backend - Node.js , Express , Socket.io 
* Deployment - Vercel , Render
* Additional Tools - Vercel v0 , Cursor
